# 基础算法 ---- 快速排序

作者：lyang（未经允许，请勿转载）

快排主要思想便是分治。

什么是分治？

  1. 分成子问题
  2. 递归处理子问题
  3. 子问题合并

而这里我大概总结了一下，便是找点排序再组合了。

什么是找点？找点就是在一个区间内随便找一个分界点，一般是中点。


...

int x=l+r>>1;//l为左边界，r为右边界，x为分界点

那么找到这么一个分界点是要干什么？

找到这么一个分界点是为了将区间分为两半

然后再调整区间：让小于等于 x 的数在左区间，大于等于 x 的数在右区间

最后在递归处理两个区间，两个区间分别排序，再组合（即两个区间再合在一起）一下便可。

快排思想图：

这里给出一个模板。


...

void quick_sort(int q[], int l, int r)
{
    if (l >= r) return;//首先判断一下，如果只有一个数字，就返回

    int i = l - 1, j = r + 1, x = q[l + r >> 1];//确定分界点x，由于是从下一位开始，所以要设l-1，r+1
    while (i < j)//设一个循环确定小于x的在左，大于x的在右
    {
        do i ++ ; while (q[i] < x);//用指针从左到右排除，如果有大于x的便会跳出循环，假设一个q[2]大于x
        do j -- ; while (q[j] > x);//用指针从右到左排除，如果有小于x的便会跳出循环，假设一个q[6]小于x
        if (i < j) swap(q[i], q[j]);//交换q[2],q[6],将小的放到左区间，大的放到右区间，直到i与j相遇为止
    }
    quick_sort(q, l, j), quick_sort(q, j + 1, r);//归并处理左，右区间
}

问题一：

如何用指针指向这串数字？


...

for(int i=0;i<n;i++) scanf("%d",&q[i]);//用for循环输入这串数字即可，数组是特殊的指针，可以精确定位数字的“位置”，那为什么用scanf呢？因为scanf比cin运行更快，可以大大提升效率

问题二：

快速排序有哪些优点呢？

正如它的名字一样，快速排序优点就是快，比起冒泡排序，一些题目它能有效的减少运行时间。

这种划分方式在平均情况下时间复杂度为 O (nlogn) O (nlog⁡n)，空间复杂度为 O (logn) O (log⁡n)。最坏情况为数组已排好序或者数组中的数都相等，此时每次划分只会将数组的长度减少 1，会递归 nn 次，导致时间复杂度为 O (n2) O (n2)，空间复杂度为 O (n) O (n)。

这里再给一下从大到小的模板


...

void quick_sort(int q[], int l, int r)
{
    if(l >= r) return;

    int i = l - 1, j = r + 1, x = q[l + r >> 1];
    while(i < j)
    {
        do i++; while(q[i] > x); // 这里和下面
        do j--; while(q[j] < x); // 这行的判断条件改一下
        if(i < j) swap(q[i], q[j]);
    }
    quick_sort(q, l, j), quick_sort(q, j + 1, r);
}
